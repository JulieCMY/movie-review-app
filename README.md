## How to Run Locally

**Clone this Project**

```
> $ git clone https://git.fdmgroup.com/Mengyu.Chen/movie-reviews-app.git
> $ cd movie-reviews-app
```

**Deploy Server**
Use maven

```
> $ mvn spring-boot:run
```

**Run Client**
```
> $ cd movie-reviews-client
> $ npm start
```
